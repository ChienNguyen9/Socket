On Linux OS
------------------

***Execute server first***

open terminal(1) server side
- type "g++ server.cpp -o server"
- type "ls" check to see if "server" is there
- type "./server"


open terminal(2) client(s) side
- type "g++ client.cpp -o client"
- type "ls" check to see if "client" is there
- type "./client"
- type "userID" if is it on the file then it will return
  the public key
